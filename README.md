# plusコマンド      
標準入力から読み込んだ数字を足し算する。
# minusコマンド     
標準入力から読み込んだ数字を引き算する。
# multiplyコマンド　
標準入力から読み込んだ数字をかけ算する。
# divideコマンド　　
標準入力から読み込んだ数字を割り算する。1
