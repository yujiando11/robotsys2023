# 1. plusコマンド      
標準入力から読み込んだ数字を足し算する。
# 2. minusコマンド     
標準入力から読み込んだ数字を引き算する。
# 3. multiplyコマンド　
標準入力から読み込んだ数字をかけ算する。
# 4. divideコマンド　　
標準入力から読み込んだ数字を割り算する。

# インストール方法・使い方
* 以下の通りにplusコマンドを作成する。
	* ①Ubuntsを開く
	* ②ホームディレクトリで「mkdir robotsys2023」とディレクトリを作成する。
	* ③そのディレクトリ内で「vi plus」とファイルを作成する。
	* ④そのファイル内に私のリポジトリのplusコマンドに書かれた内容をそのままコピペし、保存する。
	* ⑤「chmod +x plus」と実行権限を付与させる。
	* ⑥実行する。
* 上記と同様にminusコマンド、multiplyコマンド、divideコマンドもそれぞれ作り、実行する。
